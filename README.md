# Animalchain

Animalchain ist ein Tierketten-Spiel für den Browser.

## Aktueller Stand

Diese Version ist ein fertiges HTML/CSS/JavaScript-Interface ohne echte Online-Datenbank.

Enthalten:

- Startseite
- Gastnamen
- Lobby erstellen
- Lobby-Code beitreten
- Testspieler hinzufügen
- lokaler Spielmodus
- Tierliste lokal erweitern
- vorbereitet für spätere Supabase-Anbindung

## Dateien

```text
index.html
style.css
app.js
README.md
```

## Lokal starten

Einfach `index.html` im Browser öffnen.
